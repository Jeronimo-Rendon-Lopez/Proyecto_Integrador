Este es un párrafo de ejemplo para el proyecto integrador.

Este proyecto tiene como objetivo demostrar la capacidad de integrar conocimientos y habilidades adquiridos en el curso. Se espera que el proyecto sea una muestra de creatividad, innovación y aplicación práctica de los conceptos aprendidos. A través de este proyecto, se busca fomentar el trabajo, la resolución de problemas y la comunicación efectiva. El proyecto integrador es una oportunidad para aplicar lo aprendido en un contexto real y desarrollar habilidades que serán útiles en el futuro profesional.

En este proyecto se va a ver lo que e aprendido en este tiempo de aprendizaje.

Hemos aprendido a usar hojas de estilo, lo basico para hacer texto con los (h1, h2, h3, etc.) y tambien a como trabajar con imagenes y enlaces.
